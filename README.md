# AI-Powered Space Debris Detection and Collision Prediction System

## Overview

This project develops an AI system capable of detecting orbital debris from satellite imagery and predicting potential collision risks with operational satellites.

The system combines computer vision, deep learning, and orbital analytics to support space situational awareness.

## Key Features

* Satellite image debris detection
* Orbital trajectory prediction
* Collision risk analysis
* Interactive 3D orbit visualization dashboard

## Technologies

* Python
* PyTorch
* OpenCV
* YOLO Object Detection
* Plotly
* Streamlit

## Project Architecture

Satellite Images
→ Deep Learning Debris Detector
→ Orbit Prediction Model
→ Collision Risk Analyzer
→ Visualization Dashboard

## Applications

* Space traffic monitoring
* Satellite safety systems
* Orbital debris research
* Aerospace AI development

## Repository Structure

* notebooks – experimentation and model training
* models – trained AI models
* dashboard – visualization interface
* docs – technical documentation
